# 3D-fractal-openGL

Library :
pip install glfw
pip install PyOpenGL
pip install pyrr
pip install numpy


Commandes : 
python main.py

choisir 1 pour le projet cpu, puis choisir n le nombre d'iterations
OU
choisir 2 pour le projet gpu

Parametres changeables pour gpu_proj (dans le fonction draw de la class colorShader): 

- SCALE
- C_X
- C_Y
- C_Z
- ROT1_X
- ROT1_Y
- ROT1_Z 
- ROT2_X
- ROT2_Y 
- ROT2_Z
